# SPA — React + TypeScript + Vite

Одностраничное приложение на React, TypeScript и Vite.

## Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка
npm run build

# Превью сборки
npm run preview
```

## Стек

- **React 18** — UI
- **TypeScript** — типизация
- **Vite** — сборка и dev-сервер
